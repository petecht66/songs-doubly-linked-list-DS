# songs-doubly-linked-list-DS
Date Submitted: October 8, 2024
Last Updated on GitHub: April 3, 2026

This repository contains my complete work for the linked list project of CISC220: Data Structures at the University of Delaware College of Engineering Department of Computer and Information Sciences. I completed this course in the Fall of 2024, with Professor Leighanne Hsu being the course instructor. The purpose of this course was to introduce students to topics such as data type abstraction, recursion, arrays, stacks, linked lists, dynamic storage management, trees, graphs, tables, sorting, searching, and more.

This specific project dealt with a doubly linked list, a key concept for the course. The objective of this project was for students to write methods for a doubly linked list to be used in a playlist class. I created the following methods for this assigment: push, printList, remove, *pop, moveUp, moveDown, makeRandom, listDuration, and makeNewList. Then I tested these methods with the two lists of songs, as requested for this assignment.

Three folders were created for this repository. The 'header-source-files' folder contains the four C++ header source files given to students from Professor Hsu. The 'song-lists' folder contains two lists of songs that were used in this project. Finally, the 'source-files' folder contains the five C++ source files that I had to complete for this project.
